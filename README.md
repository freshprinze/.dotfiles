# Dot Files
This repository hosts the dotfiles. The files are manged by GHU Stow.

# Requirements
- GNU Stow

# Setup
1. Clone .dotfile repository

  ```
  git clone git@github.com:freshprinze/.dotfiles.git 
  ```

2. Re-link files

  ```
  cd .dotfiles
  .bin/sync-dotfils
  ```
